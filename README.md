# Customer Behavior Analysis

## Project Objective
Analyze customer transactions and behavior to identify customer segments, purchase patterns, and churn risks.

## Techniques Used
- Data Cleaning
- Feature Engineering
- RFM Analysis
- K-Means Clustering
- Data Visualization

## Tools
- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-Learn

## Key Findings
- Identified loyal customers
- Detected churn-risk customers
- Analyzed monthly purchase trends

## Recommendations
1. Loyalty rewards
2. Personalized marketing
3. Churn prevention campaigns
4. Seasonal promotions
5. Product recommendations
